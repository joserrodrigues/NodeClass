## Início
  


### Installation
1. Clone o repositório

```sh

git clone https://github.com/joserrodrigues/NodeClass.git

```

2. Instale os pacotes NPM

```sh

npm install

```

4. Altere a string de conexão no arquivo App.js


3. Inicie o servidor

```sh

npm start

```
